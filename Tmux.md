# Tmux

## Sessions

Create => `tmux new -s {name}  
Attach => `tmux attach -t {name}`
Close => `tmux kill-session -t {name}`

`tmux ls`  Shows all terminals

## Split

- Vertical Split: Ctrl+b then %.
- Horizontal Split: Ctrl+b then ".
- Switch Panes: Ctrl+b + Arrow Keys.
- Close Pane: Ctrl+b then x (then y to confirm).
- Toggle Layout: Ctrl+b then Space
